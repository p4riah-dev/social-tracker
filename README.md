# social-tracker

Automated social media presence detection and monitoring tool.

## Features
- Multi-platform scanning
- Username enumeration
- Activity detection
- Profile monitoring
- Export results to JSON

## Usage
```bash
python3 tracker.py -u <username>
```

## Examples
```bash
python3 tracker.py -u johndoe
python3 tracker.py -u johndoe --export results.json
```

## Requirements
- Python 3.x
- requests
- beautifulsoup4
